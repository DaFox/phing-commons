 5:  * Ein Universally Unique Identifier (UUID) ist ein Standard für Identifikatoren,
 6:  * der in der Softwareentwicklung verwendet wird.
 7:  * Er ist von der Open Software Foundation (OSF) als Teil des Distributed Computing Environment (DCE) standardisiert.
 8:  * Die Absicht hinter UUIDs ist, Informationen in verteilten Systemen ohne großartige zentrale Koordination eindeutig kennzeichnen zu können.
 9:  *
10:  * Obwohl die Eindeutigkeit für generierte UUID nicht garantiert ist,
11:  * ist die Gesamtzahl der zufällig generierten UUIDs (Version 4) mit 2^122 = 5,3169 * 10^36 so groß,
12:  * dass die Wahrscheinlichkeit der Erzeugung zweier gleicher UUIDs sehr klein ist.
13:  * Daher können UUIDs beliebig ohne zentrales Kontrollorgan erzeugt und zur Kennzeichnung eingesetzt werden,
14:  * ohne relevante Gefahr zu laufen, dass dieselbe UUID für etwas anderes verwendet wird.
15:  * Mit UUID markierte Informationen können somit später in einer einzigen Datenbank zusammengeführt werden,
16:  * ohne Bezeichnerkonflikte auflösen zu müssen.
17:  *
18:  * @package Lemmy
19:  * @author  Jan Thoennessen <jan.thoennessen@googlemail.com>
20:  */

23:     /**
24:      * erstellt eine 36 stellige uuid nach dce standard
25:      *
26:      * @return string
27:      */
